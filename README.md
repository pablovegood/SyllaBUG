# SyllaBUG: Comparador de Guías Docentes

Herramienta de control de versiones de bibliografías para asignaturas de la Universidad de Granada. 

Este repositorio cuenta con programas para almacenar las bibliografías de guías docentes de años anteriores (formato PDF, ejemplo -> https://grados.ugr.es/sites/grados/default/public/guias-firmadas/2024-2025/2421117.pdf) y del año actual (HTML, ejemplo -> https://derecho.ugr.es/docencia/grados/graduadoa-derecho/derechos-humanosigualdad-y-sistemas-proteccion/11/guia-docente).
